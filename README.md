    ## 项目简介
    本项目主要是对安卓技术的合计说明，包括算法、控件、设计模式、hook等。旨在为初高级安卓和Java开发者提供一些问题的解答、技术难点的分析、新技术的学习等。后续会不断更新相关的技术源码。本项目在CSDN有链接跳转。
