### 排序算法
### 关于时间复杂度
    O(n2):各类简单排序：直接插入、直接选择和冒泡排序
    O(nlog2n):快速排序、堆排序和归并排序
    O(n1+§):§ 是介于 0 和 1 之间的常数。 希尔排序
    O(n):基数排序，此外还有桶、箱排序。
### 关于稳定性
    稳定的排序算法：冒泡排序、插入排序、归并排序和基数排序
    不是稳定的排序算法：选择排序、快速排序、希尔排序、堆排序
### 名词解释：
    n：数据规模
    k："桶"的个数
    In-place：占用常数内存，不占用额外内存
    Out-place：占用额外内存
    稳定性：排序后 2 个相等键值的顺序和排序之前它们的顺序相同
